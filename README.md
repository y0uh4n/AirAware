# AirAware

AirAware is a clean and responsive weather dashboard that shows current weather, air quality, UV index, and short-term forecast data for any city using the Open-Meteo API.

## Features

- Search weather by city name
- Current temperature and “feels like” temperature
- Humidity, wind speed, and pressure
- Air Quality Index (AQI)
- Main pollutant display
- UV index with simple label
- Smart daily recommendations based on weather conditions
- 12-hour horizontal forecast
- Dark / light theme toggle
- Responsive layout for desktop and mobile
- Built with plain HTML, CSS, and JavaScript

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Open-Meteo Geocoding API
- Open-Meteo Weather Forecast API
- Open-Meteo Air Quality API
